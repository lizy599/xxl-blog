    # 403 错误

    文件协议请求数据成功，网络协议请求 403

    去掉请求的 referrer 请求头，即可解决

    <meta name="referrer" content="never" />

    # Set、Map

    Set是一种集合

    ​ 无序的，相关联的，且不重复(根据这个特性可以用来去重)的元素组成的组合

    // Set数组去重
let arr = [1,1,2,2,3,3] // 需要去重的数组
let newArr = [... new Set(arr)] // 去重后的数组

    Map是字典

    ​ 一些元素的集合，每一元素有一个key，不同元素的key各不相同(数组对象去重单个键)

    // 需要去重的数组对象
let arrObj = [
    { name: "小红", id: 1 },
    { name: "小橙", id: 1 },
    { name: "小黄", id: 4 },
    { name: "小绿", id: 3 },
    { name: "小青", id: 1 },
    { name: "小蓝", id: 4 }
];
// map的空实例对象
let map = new Map();
// 遍历
for (let item of arrObj) {
  // 判断是否含有id这个键
    if (!map.has(item.id)) {
      // 没有就设置这个id为键，值为当前遍历的对象
        map.set(item.id, item);
    };
};
// 去重后的数组对象
let newArrObj = [...map.values()];
