    # 搭建个人chat-gpt

    # 准备

    • 国外服务器，最好是内存2G以上的
    • 推荐服务器系统CentOS 7.9 64位，主要是操作简单
    • 一个属于自己的open-ai-key(某宝有，自己注册费时费力)

    # 步骤

    推荐使用宝塔面板，可视化操作相对简单

    宝塔面板官网地址 https://www.bt.cn/new/download.html

    • 1.进入宝塔面板找到这个页面

    • 2.填写对应信息，安装默认配置即可，安装完后会返回对应的宝塔地址端口号为8888和账号密码

    • 如果进不去，可能是服务器8888端口没有放开，需要去服务器进行配置

    • 3.找到SSH终端进入

      如果中途需要确认y/n，一律yes

      SSH 连接服务器后，通过命令安装 Node v16 版本：

      cd /usr/local
wget https://npmmirror.com/mirrors/node/v16.18.1/node-v16.18.1-linux-x64.tar.xz

      解压重命名到 nodejs：

      tar -xvf node-v16.18.1-linux-x64.tar.xz
mv node-v16.18.1-linux-x64 nodejs
rm -rf node-v16.18.1-linux-x64.tar.xz

      创建软连接：

      ln -s /usr/local/nodejs/bin/node /usr/bin
ln -s /usr/local/nodejs/bin/npm /usr/bin
ln -s /usr/local/nodejs/bin/npx /usr/bin

      查看版本：

      node -v  //v16.18.1

      如果能正常打印版本号，说明 Node 安装成功

      安装 PNPM 并创建软连接：

      npm install -g pnpm
ln -s /usr/local/nodejs/bin/pnpm /usr/bin
ln -s /usr/local/nodejs/bin/pnpx /usr/bin

      安装 yum-utils 包：

      yum-config-manager \
    --add-repo \
    http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

      更新 yum 软件包索引：

      yum makecache fast

      安装 docker-ce（社区版）：

      yum install docker-ce docker-ce-cli containerd.io

      如果可以查看具体的版本信息，说明 Docker 安装成功。

      docker version

      最后将 Docker 启动：

      systemctl start docker

      我找了一个比较火的 ChatGPT Web 开源项目（star 8k），链接：https://github.com/Chanzhaoyu/chatgpt-web。

      首先安装 git：

      yum install git

      下载项目（切换目录到根目录下）：

      git clone https://github.com/Chanzhaoyu/chatgpt-web.git

      # 填写秘钥

      将 chatgpt-web/service/ 目录下文件.env.example 改名为.env，并修改文件中的 OPENAI_API_KEY 值，改为你的key值（sk-开头那个），注意不需要加引号。

      # OpenAI API Key - https://platform.openai.com/overview
OPENAI_API_KEY=your_api_key

      在 chatgpt-web/service/ 目录下安装后端依赖：

      pnpm install

      在 chatgpt-web/ 目录下安装前端依赖：

      pnpm bootstrap

      在 chatgpt-web/ 目录下执行 build 操作：

      docker build -t chatgpt-web .

      build 成功后就可以运行项目啦，记得将 your_api_key 替换成你的 key：

      docker run --name chatgpt-web -d -p 3002:3002 --env OPENAI_API_KEY=your_api_key chatgpt-web

      运行成功后访问 https://ip:3002 就可以访问 ChatGPT Web 页面啦，Nice~

      如果运行成功没有页面，同样有可能是端口没放开
