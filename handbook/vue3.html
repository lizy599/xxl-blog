    # 响应式

    普通数据类型使用ref

    复杂数据类型使用reactive

    底层通过Proxy和Reflect实现

    new Proxy (代理对象,{
  get(target,propName){
    return Reflect.get(target,propName)
  }
})

    toRef和toRefs将对象里面的属性转化为响应式，方便模板里面使用

    toRaw将响应式对象转化为普通对象

    自定义ref

    customRef((track,trigger)=>{
  return {
    get() {},//需要追踪变化track
    set(newValue) {} // 需要通知Vue重新解析模板trigger
  }
})

    自定义事件

    const $emits = defineEmits(['emitName',...])
$emits('emitName',传递的东西)

    Vue3 v-model实现

    <xxx v-model="val" /> 封装input组件
  const val = ref('')
  const props = defineProps({
  modelValue: String
})
	const $emits = defineEmits(['modelValue'])

<input @input="handleInput" />
  const handleInput = (e)=>{
    $emits('modelValue',e.target.value)
  }

    # 侦听器

    watch

    watch(prop,(newVal,oldVal)=>{},{immediate:true})
watch([prop1,prop2],(newVal,oldVal)=>{
  // newVal和oldVal也是数组
},{immediate:true})

    监测深层数据也能实现，不需要配置deep:true，监测对象时，oldVal和newVal值一样

    监测对象的某个属性

    watch(()=>obj.prop,()=>{})

    特殊情况，当对象的某个属性是引用类型，修改内层的东西需要开启deep:true

    watchEffect

    类似computed，但computed注重返回的结果，watchEffect注重过程，两者都是内部依赖发生变化，就会触发执行
